# Arg Tester

Send loads of arguments to a program to test it responses.

## Usage

```bash
java -jar main.jar [COMMAND] [ARGS]
```

## Example

```bash
java -jar main.jar "echo {1} {2}" "args.1.txt,args.2.txt"
```
